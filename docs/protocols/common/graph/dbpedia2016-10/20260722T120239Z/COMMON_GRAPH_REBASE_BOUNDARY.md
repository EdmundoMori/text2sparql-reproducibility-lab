1. Modern **immutable** snapshot pin

3. Gold query re-execution
4. Recalculated answers

6. Defect adjudication
7. Explicit rebase version label
8. Results stored **separately** from native historical evaluation
9. **No** numeric comparison presented as historical reproduction
10. **No** selective repair of individual questions via rebase
