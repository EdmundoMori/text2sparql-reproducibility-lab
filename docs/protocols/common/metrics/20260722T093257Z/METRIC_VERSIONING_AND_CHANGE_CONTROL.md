# METRIC_VERSIONING_AND_CHANGE_CONTROL

**metric_contract_version:** `0.1.0-documentary`

- Redacción sin efecto: PATCH
- Canonicalization/fórmulas: MAJOR
- Nuevas métricas sin alterar existentes: MINOR
- Cada versión tiene hash; benchmark ↔ una sola versión
- Cambio post-ejecución → invalidar o recalcular todos
- No mezclar resultados de versiones distintas
